# raycast-keyboard-maestro-extension

[Raycast](https://www.raycast.com/store) extension for listing Keyboard Maestro macros.

Originally created by Eric ([@eluce2](https://github.com/eluce2))

## Changes introduced:

### 2023-10-29

- New preferences page:
  - Filter for macro groups, including regex support.
  - Display keyboard shortcuts and string triggers for macros.
  - Display macro icons.
- Few minor optimizations and fixes.
